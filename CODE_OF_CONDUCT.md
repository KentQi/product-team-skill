# 社区行为准则

## 我们的承诺

为了营造一个开放、友好、包容的环境，作为本项目的贡献者和维护者，我们承诺：无论年龄、体型、残疾、种族、性别认同与表达、经验水平、国籍、个人形象、种族、宗教、性取向如何，参与本项目的每一个人都能获得无骚扰的体验。

## 我们的标准

有助于创建积极环境的行为包括：

- 🤝 使用友好和包容的语言
- 👂 尊重不同的观点和经验
- 📝 优雅地接受建设性批评
- 🌱 专注于对社区最有利的事情
- 💝 对其他社区成员表示同理心

不可接受的行为包括：

- ❌ 使用性化的语言或图像，以及不受欢迎的性关注或调情
- ❌ 侮辱/贬损性评论、人身攻击或政治攻击
- ❌ 公开或私下骚扰
- ❌ 未经明确许可，发布他人的私人信息（如物理地址或电子地址）
- ❌ 在专业环境中可被合理视为不当的其他行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并应对任何不可接受行为的实例采取适当且公平的纠正措施。

项目维护者有权和责任删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，临时或永久地禁止任何他们认为行为不当、威胁性、冒犯性或有害的贡献者。

## 适用范围

本行为准则适用于本项目的所有空间，包括但不限于：

- GitHub 仓库（Issues、Pull Requests、Discussions）
- 项目相关的线上/线下活动
- 代表项目或在项目中进行的任何形式的交流

## 举报

如果发生不可接受的行为，或你有任何疑虑，请通过以下方式联系项目维护者：

- **GitHub Issues**：[@KentQi](https://github.com/KentQi)
- **邮件**：通过 GitHub 个人资料获取

所有投诉都将被及时审查和调查，并将根据具体情况采取必要且适当的回应。项目团队有义务对举报者保密。
